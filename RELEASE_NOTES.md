# YTDownloader4K - Release Notes

## How to Use (Windows).

1. **Download** `YTDownloader4k.exe` from the latest release
2. **Double-click** the .exe file to run
3. **No installation needed!** Everything works immediately

## What's Included

✅ **Completely Standalone** - No dependencies required  
✅ **FFmpeg Bundled** - Audio/video processing included  
✅ **Python Runtime** - No Python installation needed  
✅ **yt-dlp** - Latest YouTube download engine  
✅ **Auto-Update** - Checks for new versions on startup

## System Requirements

- **Windows 7, 8, 10, or 11** (32-bit or 64-bit)
- **512 MB RAM** minimum
- **Internet connection** required for downloads
- **No admin rights** needed

## Features

- Download videos in **maximum quality** (up to 4K/8K)
- Download videos in **720p** for smaller file sizes
- Extract **audio only** and convert to MP3
- Smart download management
- Geo-restriction bypass attempts
- Automatic retry on failures

## First Time Setup

**There is no setup!** Just run the .exe and start downloading.

The app will:
1. Check for updates automatically
2. Update yt-dlp library if needed
3. Open the main download window

## Where Are My Downloads?

All files are saved to your **Downloads** folder by default:
- `C:\Users\YourUsername\Downloads\`
